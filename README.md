# Book Tracker

**Автор:** Иван Иванов  
**Язык разработки:** Python 3 (Tkinter)

## Описание программы
«Book Tracker» — это настольное GUI-приложение, предназначенное для ведения списка прочитанных книг. Программа позволяет структурировать личную библиотеку, сохранять историю чтения и быстро находить нужную информацию с помощью гибких фильтров. 

### Основные возможности:
* **Учет книг:** Хранение названий, авторов, жанров и количества страниц.
* **Валидация:** Защита от пустого ввода и проверка числового формата для страниц.
* **Фильтрация:** Поиск книг по совпадению жанра и отбор книг по минимальному количеству страниц (например, более 200).
* **Постоянное хранение:** Автоматическое сохранение базы данных в локальный файл `books.json` и чтение из него при запуске.

## Инструкция по запуску

### Требования
Для работы приложения необходим установленный интерпретатор Python версии 3.x. Дополнительные библиотеки устанавливать не нужно (`tkinter` и `json` входят в стандартную сборку).

### Запуск приложения
1. Скачайте исходный код или клонируйте репозиторий.
2. Откройте терминал/командную строку в папке с проектом.
3. Выполните команду:
   ```bash
   python main.py
   ```

## Примеры использования (Тест-кейсы)

### Тест 1: Добавление новой книги
1. Заполните поля: 
   * Название: *Преступление и наказание*
   * Автор: *Фёдор Достоевский*
   * Жанр: *Классика*
   * Страниц: *600*
2. Нажмите кнопку **«Добавить книгу»**.
3. *Результат:* Книга отобразилась в таблице снизу, поля ввода очистились. Создался файл `books.json`.

### Тест 2: Проверка валидации на число строк
1. Заполните текстовые поля, а в поле «Страниц» введите текст: *много*.
2. Нажмите кнопку **«Добавить книгу»**.
3. *Результат:* Появится всплывающее окно с ошибкой «Количество страниц должно быть целым числом!». Данные в таблицу не занесутся.

### Тест 3: Фильтрация данных
1. Добавьте три книги с разным количеством страниц (например: 150, 300, 450).
2. В панели фильтрации в поле «Мин. страниц» введите `200` и нажмите **«Применить»**.
3. *Результат:* В таблице останутся только две книги (с 300 и 450 страницами). По кнопке **«Сбросить»** отобразятся снова все записи.
