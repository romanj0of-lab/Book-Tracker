## Описание программы
«Book Tracker» — это настольное GUI-приложение для управления списком прочитанных книг. Оно позволяет структурировано хранить информацию о домашней или личной библиотеке, осуществлять фильтрацию по числовым критериям и жанрам, а также автоматически синхронизировать данные с локальным хранилищем.

### Основные возможности и особенности:
* **Удобный GUI-интерфейс:** Поля ввода текстовых значений, функциональные кнопки действий и интерактивная таблица для вывода данных.
* **Полноценная валидация:** Программа блокирует пустой ввод и гарантирует, что количество страниц может быть только целым положительным числом.
* **Фильтрация в реальном времени:** Возможность отбора книг по частичному совпадению жанра и по минимальному количеству страниц (например, `>= 200`).
* **Отказоустойчивость (Обработка исключений):** Приложение защищено от сбоев файловой системы, ошибок отсутствия доступа к файлам и повреждения структуры `books.json`.
* **Автосохранение:** База данных автоматически подгружается при старте программы и обновляется в JSON-файле при добавлении записей.

## Инструкция по запуску

### Требования к окружению
Для запуска необходим установленный интерпретатор **Python версии 3.6 или выше**. Стандартная библиотека содержит все необходимые модули (`tkinter`, `json`, `os`), поэтому дополнительные установки через `pip` не требуются.

### Инструкция по запуску
1. Склонируйте репозиторий или скачайте архив с кодом проекта.
2. Откройте терминал (или командную строку Windows / Git Bash) в директории проекта.
3. Выполните стандартную команду запуска:
   ```bash
   python main.py
   ```

## Примеры использования (Тест-кейсы)

### Тест 1: Успешное добавление книги в список
* **Действие:** Заполните форму данными: `Название` = "Капитанская дочка", `Автор` = "Александр Пушкин", `Жанр` = "Повесть", `Страниц` = "160". Нажмите **«Добавить книгу»**.
* **Результат:** Книга успешно заносится в таблицу на экране. Поля формы очищаются. Рядом с `main.py` появляется файл `books.json` со структурированным текстом книги.

### Тест 2: Проверка обработки некорректного ввода (Валидация)
* **Действие:** Оставьте поле `Название` пустым или впишите в поле `Страниц` текст "двести страниц". Нажмите кнопку добавления.
* **Результат:** Приложение прерывает обработку, не добавляет мусорные данные в таблицу и выводит информационное диалоговое окно `messagebox.showerror` с описанием ошибки ввода.

### Тест 3: Работа фильтров
* **Действие:** Добавьте 3 любые книги (например, на 100, 250 и 400 страниц). В панели фильтрации в поле «Страниц (от)» введите `200` и нажмите **«Применить фильтр»**.
* **Результат:** В интерактивной таблице мгновенно скрывается книга со 100 страницами. После клика на **«Сбросить»** исходная таблица отображается полностью.
