# Генератор случайных цитат (Quote Generator)

**Автор:** Арина
**Стек технологий:** Python с нуля: первые шаги в программировании (начальный уровень) офлайн

## Описание программы
Приложение предназначено для генерации случайных вдохновляющих цитат великих людей из встроенной базы данных. Каждая сгенерированная цитата фиксируется в интерактивном списке истории просмотров. История автоматически сохраняется на жесткий диск и подгружается при каждом перезапуске программы.

### Основные функции:
* **Случайный выбор:** Быстрая генерация цитат с использованием встроенного модуля `random`.
* **Фильтрация:** Поиск по накопленной истории сгенерированных цитат по имени автора или конкретной тематике (например: "Философия", "Успех").
* **Полноценный GUI:** Удобные текстовые поля ввода, кнопки вызова событий и таблица вывода информации (Treeview).
* **Валидация:** Проверка на пустые строки при поиске и фильтрации для исключения сбоев.
* **Работа с данными (JSON):** Надежное сохранение и загрузка истории просмотров в файл `quotes_history.json` с обработкой исключений `try-except`.

## Инструкция по запуску

### Требования
Для работы программы требуется установленный интерпретатор **Python версии 3.6 или выше**. Дополнительные внешние библиотеки устанавливать не нужно.

### Запуск приложения
1. Откройте терминал или командную строку (например, Git Bash) в директории проекта.
2. Запустите скрипт командой:
   ```bash
   python main.py
   ```

## Примеры использования (Тест-кейсы)

### Тест 1: Генерация цитаты
1. Нажмите кнопку **«Сгенерировать случайную цитату»**.
2. В верхней панели отобразится случайная цитата и её автор, а в нижней таблице («История») появится новая строка.
3. В корне проекта автоматически сгенерируется файл `quotes_history.json`.

### Тест 2: Фильтрация по автору
1. Нажмите кнопку генерации 5–10 раз, чтобы наполнить историю.
2. В панели фильтрации в поле «Автор» введите `Шекспир`. Нажмите **«Применить фильтр»**.
3. В таблице останутся только цитаты Уильяма Шекспира. Нажмите **«Сбросить»** для возврата к полной истории.

### Тест 3: Проверка валидации пустого ввода
1. Оставьте поля «Автор» и «Тема» пустыми.
2. Нажмите кнопку **«Применить фильтр»**.
3. Программа прервет операцию и покажет предупреждение: *«Введите текст в поле Автор или Тема для фильтрации!»*. Сбоя или вылета приложения не произойдет.
