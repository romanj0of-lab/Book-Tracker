# Expense Tracker — Трекер личных расходов

**Автор:** Сайфутдинова Арина Фаритовна  
**Языковая среда:** Python 3 (Tkinter GUI)

## Описание программы
«Expense Tracker» — это настольное GUI-приложение, созданное для ведения персонального финансового учета и мониторинга личных расходов. Программа помогает классифицировать траты по различным категориям, отслеживать финансовые потоки во времени, автоматически вычислять итоговую сумму издержек за конкретный выбранный период и осуществлять фильтрацию накопленной базы данных.

### Ключевые возможности проекта:
* **Графический интерфейс пользователя (GUI):** Форма ввода параметров, выпадающие списки категорий и таблица для вывода финансовых записей.
* **Строгая встроенная валидация:** Защита от пустых значений. Контроль числовых полей (сумма должна быть строго положительным числом `> 0`). Контроль временных полей (ввод дат допускается только в формате `ДД.ММ.ГГГГ`).
* **Динамический подсчёт сумм:** Приложение автоматически пересчитывает итоговую сумму затрат в реальном времени при применении любых фильтров за выбранный период.
* **Сериализация данных в JSON:** Локальная синхронизация с файлом `expenses.json` при добавлении записей и автозагрузка истории при запуске программы.

## Запуск

### Требования к системе
Для работы необходим установленный интерпретатор **Python версии 3.6 или выше**. Дополнительные внешние библиотеки устанавливать не нужно.

### Инструкция по запуску
1. Откройте терминал, командную строку или консоль Git Bash внутри папки с файлами проекта.
2. Выполните следующую команду для автоматического старта приложения:
   ```bash
   python main.py
   ```

## Примеры использования (Тест-кейсы)

### Тест 1: Регистрация нового расхода
1. Внесите в элементы интерфейса параметры: Сумма = `1250.50`, Категория = `Еда`, Дата = `26.05.2026`.
2. Нажмите экранную кнопку **«Добавить расход»**.
3. *Результат:* Расход зафиксирован строкой в таблице, поле суммы очистилось, а текстовый индикатор внизу обновил общую сумму трат.

### Тест 2: Проверка обработки ошибочного ввода
1. Введите в поле суммы буквенный текст (например, *тысяча рублей*). Нажмите **«Добавить расход»**.
2. *Результат:* Алгоритм заблокирует транзакцию, данные не попадут в отчет, и программа отобразит диалоговое окно с уведомлением: *«Сумма должна быть положительным числом!»*.
